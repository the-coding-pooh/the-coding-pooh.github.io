Virtual OI BOI
Virtual : BOI 2019
10月 29, 2024

前言

Written by pooh at 2024/10/29, 2024/10/30
要打北市賽了，小緊張，這兩天用公假來打 BOI，主要其實是想穩心態，但打了其實可能心態更不穩。

總之我獻祭了我的獎金想要北市賽打很好，希望 Ina 會保佑我。

Day 1

一些廢話

在費曼 vir 的，最喜歡公假了。

開始暴雷

題目

請自行閱讀

對不起我懶 QAQ

賽中過程

0:00

pA 看到就會某種把 個 bit 當一個去做，不知道會有幾分。
然後 pB 感覺就會 dp 亂做到第二個子題，可能用個 bitrick 搞搞。
pC 感覺就是樹剖亂搞一搞可做的東西，再看看。

實際分數: 0 / 0 / 0
精神分數: ? / 66 / 36

0:33

稍微想了一下以後就發現 pB 的轉移可以 bitset 亂做就好，然後就去寫了，先把沒有問號的丟上去驗一下，就拿到 29 分了。

實際分數: 0 / 29 / 0
精神分數: ? / 100 / 36

1:01

把問號的東西處理了一下，然後範測 WA 了以後才發現他是問你有幾個最終狀態，所以又再稍微處理了一下，然後就過了。

實際分數: 0 / 100 / 0
精神分數: ? / 100 / 36

1:32

稍微寫了一下那個解以後就拿到了 36.9 分，然後就溜了。

實際分數: 36.9 / 100 / 0
精神分數: 36.9 / 100 / 36

然後就去想了一下，發現 pC 我會樹剖做，但可以不用，然後又想了一下才想到可以倍增做，我真的是菜雞 QAQ。

2:32

稍微實作了一下，然後就死掉了。接著就發現我倍增的東西好像上下寫反了，de了很久都de不乾淨，索性刪掉重寫。

實際分數: 36.9 / 100 / 0
精神分數: 36.9 / 100 / 100

4:00

終於把東西都弄乾淨了，然後就 AC 了，我真的是笨 QAQ。

實際分數: 36.9 / 100 / 100
精神分數: 36.9 / 100 / 100

然後中間交了很多次 pA 分數都沒有變高，最後去看解了以後發現他真的是詭譎。

檢討

耍笨還是沒有少耍，今天的實作真的不太漂亮，但還是可以理解的程度，pA 真的太怪，pC 應該要更早寫出來的，這場真的不難，不過有策略真的有差，不管是寫爛重寫或把開始寫題的時間稍微往後讓我想的到 pB 真的都是策略有用。

最終分數 : 236.9
普通，跟我一起 vir 的 zzw 跟 cot 也差不多 (只有pB被沒有引導性的子題嗆)。

Day 2

一些廢話

Day 1 隔天一鼓作氣 vir 了，結果現在晚上心情超差，隨然成績上來講不是 vir 過最爛的，但也真的讓我覺得我的策略真的有很大需要調整的東西，還沒想清楚實際是怎麼調，但總之就是需要調。

只希望我的心理素質能夠強，不要影響到北市賽的心態。

開始暴雷

題目

我先去emo，自己看

賽中過程

0:00

pA 把式子寫下來以後就知道是背包了，亂做一下就會寫完， pB 感覺就是怪怪字串，沒想法，pC 看起來就是 fracturing search，之前精神過，不知道寫不寫的出來。

實際分數: 0 / 0 / 0
精神分數: 100 / 25 / 100

0:44

pA 很快就寫好拿到分數了，跑去做 pC a.k.a. 噩夢的開始。

實際分數: 100 / 0 / 0
精神分數: 100 / 25 / 100

1:42

pC 寫了第一個版本出來了，然後就 TLE 了。

實際分數: 100 / 0 / 0
精神分數: 100 / 25 / 100

2:20

壓了個常，做了個優化，然後就 WA 了。

實際分數: 100 / 0 / 13
精神分數: 100 / 25 / 100

2:40

找不到 bug，以一個後設的觀點來看其實那個 bug 有點蠢，就我沒想懂 fracturing search 在幹嘛而已。

然後我這個時候就在想要不要去唬爛，解果唬爛也唬得不乾脆，再以一個後設的觀點來講我如果時間剪枝一下就 68 分了，還只 WA 一筆欸。

實際分數: 100 / 0 / 13
精神分數: 100 / 25 / (68)

3:40

去把 pB 的 45 分的式子推完了，然後如果只看等號是對的那就把 lcp 改成用 z 算就可以拿到滿分了，終於可以開寫了。

實際分數: 100 / 0 / 13
精神分數: 100 / 100 / (68)

4:03

寫好以後 WA 了，完全慌掉，找不到原因，我原本對我自己實作的信心是只少 30 min 內該要寫完的 QAQ。

實際分數: 100 / 0 / 13
精神分數: 100 / 100 / (68)

4:45

花了整整 42 分鐘才發現輸出是要 0 based 的，虧我還自己幫他加一，被我自己氣死，最後完全沒有時間寫滿分解。

實際分數: 100 / 45 / 13
精神分數: 100 / 100 / (68)

檢討

講白了打完這場後我非常失望，只要我開始唬爛的乾脆一點，拿時間剪枝砸一砸，或是提早放棄改去想 pB 早點做完我都可以破 200，只能說表現得非常差，讓我不經再思考，究竟是我的策略沒有好好思考唬爛這個東西的定位讓我不知道該不該先做他，還是我只是用策略問題來催眠我自己我的實力不如反映出來的如此差勁，好讓我相信不是因為我遲鈍的實作或是緩慢的思考速度遭成的。

但反正就這樣了，北市賽希望能打好一點，策略面上比較有料的還是想想唬爛這個東西到底該是什麼樣的一個定位吧。

最終分數 : 158

兩天總分

大大暴雷

總分 : 394.9
Rank 8， Silver。
day 1 pA 基本上是通靈不到的，第一次寫 two step 能有這個分數我覺得還可以，但 day 2 真的是我沒有好好想過唬爛這個東西對我得定位來講是什麼，造成了兩題加起來損失 110 分的慘劇，就算拿到一半的分數也能有金牌，都拿到就 rank 1 了 QAQ。

晚上找了芒果討論了一下，接下來也許合理的唬爛就剛被當作正解來看待，不要想說丟到最後才做，而沒有道理的唬爛就該被往後丟吧。

只能說失望，只少北市賽該會打好了吧。
